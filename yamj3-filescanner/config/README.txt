YAMJ v3 File Scanner
~~~~ ~~ ~~~~ ~~~~~~~
This is the file scanner module for YAMJ v3

Edit the config/yamj-filescanner.properties file and insert your host and port settings in yamj3.core.host and yamj3.core.port
properties.

To use, run the filescanner.cmd file with the "-d" property to pass the root of the directories to scan.
